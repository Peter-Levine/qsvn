Welcome to QSvn

QSvn is a GUI Subversion client for Linux, UNIX, Mac OS X and Windows. It is a real client not a GUI wrapper for the command line client from Subversion.

QSvn uses a C++ Subversion wrapper developed by the Kdesvn Project. For cross plattform capability, QSvn uses the Qt Framework.

The latest version of QSvn is version 0.8.3, released on July 26, 2009.

Development

The development of QSvn is stopped. I have switched all my private repositories from Subversion to Git. So i haven't any interest in a Subversion client anymore.

The last commit is some years ago.

Open Source

QSvn is completely Open Source. As i decided to finish the development in QSvn i searched a solution for hosting the project outside my own server. So here it is.

If you are a Developer, who loves Open Source, Subversion, Qt and QSvn or even one of them, feel free to fork the project and develop the next big version of QSvn.
